# Autoencoder

**MLP-based autoencoder** for dimensionality reduction and reconstruction.

## Overview

- Encoder-decoder architecture with MLPRegressor
- Configurable encoding dimension
- Side-by-side original vs reconstructed visualization
- **Streamlit dashboard**

## Quick Start

```bash
pip install -r requirements.txt
streamlit run app.py
pytest tests/ -v
```

## Docker

```bash
docker compose up --build
```
